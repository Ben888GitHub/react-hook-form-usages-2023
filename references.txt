Common Validation Messages as examples:
- "this field is required", means you've left the field blank
- "Please enter in format DD/MM/YYY", means you've entered data in the wrong format
- "Password should be between 8 and 30 characters long and contain one uppercase letter, one number, and one special character": You're not using strict data format requirements for this field

Form Validation Importance
- Website Security (by applying strict validations input)
- Prevents Human Error
- Protects User Data (highly secure password)

Basic Built-in Validation
- Type: specific type data such as email, URL, phone number
- Required: field input is mandatory and can't be blank
- Minlength and maxlength: minimum and maximum length of characters
- Pattern: Regex Pattern
